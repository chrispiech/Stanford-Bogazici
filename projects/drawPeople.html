Three's a Crowd

Problem written by Julia Lee and Chris Piech.

Handouts: Graphics Reference
Worked Example: Target

One awesome application of computer science is using it to construct intersting custom data visualizations like this one:

With your new graphics skills, you can too! Write a GraphicsProgram that draws a person graphic at three different locations.

Details

  1. A person is composed of a head, body, and legs. The starter file (drawPerson.java) contains the default dimensions and colors.
  2. You should write a method with two parameters -- an x-coordinate and a y-coordinate -- that draws a person with a head is at those coordinates.
    private void drawPerson(int headX, int headY)
  3. You should draw a person at the following three locations:
    1. (40, 40)
    2. (300, 300)
    3. (600, 175)
